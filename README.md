# 🧩 UI Playground

A modern component playground built with **React**, **Storybook 9**, and **Tailwind CSS**.  
Designed to **prototype**, **document**, and **test** UI components in isolation with live previews, interactions, and visual theming.

---

## ✨ Features

- ⚛️ React + TypeScript component library
- 🎨 Styled with Tailwind CSS
- 📘 Auto-generated docs via Storybook
- 🧪 Interactive stories using `@storybook/testing-library`
- 💡 Shimmer loading effect showcase
- 🌓 Dark/light mode background toggle
- 🧩 Design token support & custom docs blocks
- 🔍 `play()` function support for interaction tests

---

## 📦 Tech Stack

- **React** (TypeScript)
- **Storybook 9** (Webpack 5)
- **Tailwind CSS**
- `@storybook/testing-library` — interaction testing
- `@storybook/addon-docs` — documentation
- `@storybook/addon-themes` — theme switching

---

## 🧪 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/gayathri1462/ui-playground.git
cd ui-playground

# 2. Install dependencies (Storybook 9 may require --legacy-peer-deps)
npm install --legacy-peer-deps

# 3. Start Storybook
npm run storybook
````

---

## 📤 Build & Deploy

### ➤ Build static Storybook

```bash
npm run build-storybook
```

### ➤ Deploy to GitHub Pages

```bash
npm run deploy-storybook
```
